A very simple bit of code to create an extension for Chrome that changes the default new tab to a custom image.
To use:
1. Download and extract zip
2. Place desired image in extracted folder
3. Edit newTab.html in any text editor. Replace YourImage.jpg with the name of your image.
4. Open Chrome and click the Menu button. Go to More Tools -> Extensions.
5. Click Load unpacked extension and select the extracted folder.
